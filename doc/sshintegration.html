UnionPass可以为您的Unix、Linux类型的主机的远程访问（SSH）轻松增加安全认证。

前置条件

在使用之前，你需要：

  1. 注册成UnionPass系统用户。
  2. 创建应用，获得APPID。

设置

脚本

“UnionPass移动认证服务”提供针对SSH服务的安全脚本：

  • $ curl -k 'https://www.mauth.cn/tools/ssh/unionssh' -o unionssh

安装

  • $ sudo bash unionssh install /usr/local/bin
    • 需要输入你注册服务后获得的APPID。
    • 请选择1启用两步认证（选择2则为传统的OTP模式）。

注意

执行此命令，需要您具有root权限，以及注册后创建的APPID。

  • $ sudo service ssh restart
    配置成功后，重启服务。

关联帐号

  • $ sudo /usr/local/bin/unionssh enable `whoami`
    此命令，将帮助您直接创建用户：
    • whoami参数可以为需要使用UnionPass认证服务的系统用户名称。
    • 你需要输入该帐号对应的手机号和email信息。

编辑

unionssh安装后，也可以手工编辑unionssh.conf文件。内容如下:

字段 内容
api_key 您获得的APPID
user 本地帐号:UnionPass帐号
